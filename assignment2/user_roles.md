User Roles

1. Job Seekers (Students / Intern Candidates)

Description:
Students or fresh graduates looking for internship opportunities.

Goals:

Discover relevant internships
Apply to positions
Track application status
Complete assigned internship tasks
Receive evaluations and certificates

Permissions:

Create and manage profile
Upload resume and portfolio
Search and filter internships
Apply for internships
Track application progress
Accept/reject offers
Submit internship reports/tasks
View feedback and ratings
Download internship completion certificate
2. Job Providers (Companies / Recruiters)

Description:
Organizations or recruiters offering internship opportunities.

Goals:

Post internship openings
Find suitable candidates
Manage applications
Monitor intern performance

Permissions:

Create company profile
Post, edit, and close internship listings
View applicant profiles
Shortlist/reject candidates
Schedule interviews
Send internship offers
Assign tasks/projects
Evaluate intern performance
Issue completion certificates
3. Admin

Description:
Platform administrators responsible for operations, moderation, and compliance.

Goals:

Maintain platform integrity
Manage users and content
Resolve disputes
Generate platform insights

Permissions:

Manage all users
Verify company accounts
Approve/reject internship postings
Moderate content and reports
Suspend fraudulent accounts
View platform analytics
Manage certificates and templates
Handle support requests
Configure platform settings
